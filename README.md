### 9. Staging Area 에 올라간 파일을 로컬 브랜치에 Commit 해주세요! (커밋 메시지는 자유롭게)

<details>
    <summary>힌트보기</summary>
    ... 대표는 아기사자에게 실망했다
    
    git commit -m "COMMIT MESSAGE"

</details>

### 10. 로컬 브랜치를 원격 리포지토리에 push 해서 브랜치를 퍼블리시 해주세요!

<details>
    <summary>힌트보기</summary>
    이번 한번만입니다
    
    git push --set-upstream origin <아까만들었던 브랜치 이름>
</details>
